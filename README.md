# SpeakerGUMBO

Repository for Speaker GUMBO Project



\# Speaker GUMBO



Parametric OpenSCAD program for generating 3D-printable speaker enclosures.



Multi-file design supporting sealed, ported, different drivers, bracing, etc.



\## Current Version

0.1.0 (initial public version)



\## Usage

\- Open `gumbo-main.scad` in OpenSCAD

\- Adjust parameters at the top or in included files

\- Render (F6) → Export → STL for slicing/printing



See `examples/` for ready-to-tweak configs.
